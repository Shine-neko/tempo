Contributing
------------

Tempo is an open source, community-driven project.

* **Pull Request Template** : Template header to use in your pull request description;

```markdown
| Q             | A
| ------------- | ---
| Bug fix?      | yes/no
| New feature?  | yes/no
| BC breaks?    | no
| Deprecations? | no
| Tests pass?   | yes
| Fixed tickets | #1
| License       | MIT
```
